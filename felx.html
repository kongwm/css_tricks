Flexbox 属性手册

Flexbox 属性分为两类：设在父容器上的布局指令，以及设在子元素上的自定义行为。

控制主轴方向，决定子元素的排列方式（横向或纵向）。

flex-grow

flex-shrink
空间不足时的收缩比例。默认 1 等比收缩，设为 0 则保持原始尺寸不压缩。
0（不收缩） 1 2（收缩更多）
flex-basis
子元素在主轴方向上的初始大小，在 grow/shrink 计算之前生效。
auto 200px 30% content
flex（简写）
= flex-grow flex-shrink flex-basis，推荐始终用简写代替分别写三个属性。

常用速记模板
/* 基础水平垂直居中 */
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
 
/* 等比撑满剩余空间 */
.item { flex: 1; }
 
/* 固定侧边栏 + 自适应主内容 */
.layout { display: flex; }
.sidebar { flex: 0 0 240px; }
.main   { flex: 1; }
 
/* 等高卡片（stretch 默认行为）*/
.cards { display: flex; gap: 16px; }
.card  { flex: 1; } /* align-self: stretch 为默认 */
